Móntate tu propio bot por cuatro duros

Sun 09 October 2016
By Alberto Fernández
  • YouTube

Description

Amazon Lambda y API Gateway para la infraestructura, Telepot para acceder a la API de Telegram, Amazon DynamoDB para guardar el estado.

Pocos ingredientes pero cóctel explosivo. Gracias a la expresividad que nos proporciona Python, aunque anclados al pasado por la obligación de usar la versión 2.7, en muy pocas líneas de código podemos conseguir la suficiente funcionalidad en un bot.

El resultado es un asistente para la PyConEs a la que podemos indicarle nuestros intereses para que nos recomiende charlas y talleres, crear nuestra propia agenda y que nos avise antes de que empiece cada evento a los que nos hayamos apuntado.

Durante la charla introduciremos los puntos clave para que cualquiera entienda y pueda montar su propio bot. Todo el código se hará público en Github para que otros puedan hacer sus propias versiones.

Presentación disponible en https://voodmania.github.io/pycones2016-presentacion/

http://2016.es.pycon.org/
