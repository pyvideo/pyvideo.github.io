Docker meets Python - A look on the Docker SDK for Python

Fri 12 July 2019
By Jan Wagner
  • YouTube

Description

My talk aims to introduce and have a closer look on the Docker SDK for Python.

I will cover:
- How and where to get the SDK
- How it works and how to use it in general
- Possible use-cases like: Processing Container-Logs, Testing with pytest on different Python Versions, Deploy via Python Script, etc..
For my talk, you should know what Docker is and how to use it.
A basic idea of pytest and server administration is nice to have, but not necessarily needed to follow my talk.
