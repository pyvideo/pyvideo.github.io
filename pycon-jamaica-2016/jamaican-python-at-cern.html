Jamaican Python at CERN

Fri 18 November 2016
By Dominic Mills
  • YouTube

Description

Dominic Mills presents on his work with the Large Hadron Collider. Some objectives of the project included: Integrating the Celery library to manage data received from the sensors in a distributed asynchronous manner. Integrating the Bokeh library to produce real-time plots from the asynchronous data read from the sensors. Implementing functionality to trigger alarms in case of any faulty hardware. Creating a language agnostic description of the hardware with XML.

https://www.pythonjam.org.jm/conference-2016 the PyCon Jamaica 2016 technology conference for Python software developers was held (November 17 - 18, 2016). The conference was immediately followed by the Great Python Scavenger Hunt https://www.pythonjam.org.jm/conference-2016/scavenger-race.
