使用 Pyodide 讓你在瀏覽器中直接運行 Python - 以醫療數位影像 DICOM 視覺化為例 – 康登傑 (PyCon Taiwan 2021)

Sat 02 October 2021
By Grimmer Kang
  • YouTube

Description

Day 1, 11:35-12:05

Abstract

Mozilla 開發的 Pyodide 框架是一套可以讓你在瀏覽器中撰寫 Python 程式碼以及使用科學相關常用的套件，如 NumPy, Pandas、 Matplotlib、SciPy 以及 scikit-learn。原理是把 Python 轉譯成 WebAssembly 運行在瀏覽器中。你可以在瀏覽器裡使用 JavaScript 取得 Python 計算的結果，或是跟其互動。 此演講會以實際的例子，使用 Python 來解析醫療影像 DICOM 並把結果傳回到瀏覽器上呈現出來。過程中會介紹到使用 Pyodide 的細節及優缺點、醫療影像 DICOM 格式的介紹、以及不同實作方式間的比較。 此演講適合對象 (滿足其一即可) 1. 對 JavaScript 有初步的認識或對跨平台有興趣的人，因其需要使用 JavaScript 來跟 Python 互動。 2. 對醫療數位影像 DICOM 解析或視覺化有興趣的人。
