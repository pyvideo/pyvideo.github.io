使用 Python 實現可即時互動用的遠端影音傳輸 – Milo Chen (PyCon Taiwan 2021)

Sun 03 October 2021
By Milo Chen
  • YouTube

Description

Day 2, 10:35-11:20

Abstract

在過去， Python 是電腦上非常的重要工具，更是許多人第一個學會的程式語言，並且已經在市面上擁有高普及性也成功導入各場域，例如像是物聯網應用以及AI應用。 今日，全球世界趨勢正加速變化，遠端生態圈此時正在快速形成，過去一年來的上市公司及許多獨角戰新創皆是以「遠端生態系」的方式進入新世界；影音傳輸很長見，但速度能快到即時互動的不常見，一個能滿足遠端 "即時互動需求" 的影音傳輸技術及應用，便是這個新生態系的一張入場卷。 可是你們知道嗎? 神奇 Python 也能讓我們作到。 本場演講會介紹如何使用 Python 的 AIORTC 框架來實現 一個能滿足遠端即時互動需求的影音傳輸技術及應用 1. 不知道需要有哪些背景知識嗎? 這邊會告訴你 2. 不知該從何實作起嗎? 這邊會帶你實作 3. 演講結束後怕不知從何繼續嗎 ? 這裡會留參考專案與後續聯絡管道 本演講將適合給每一位有 python 專案開發經驗的人聽， 一方面也鼓勵聽眾能在聽演講之前先初淺了解一下OSI七層網路架構。

Description

主要對象是給那些想要以 python 實現各樣即時影音互動應用的人，這些人的背景可以是無任何python框架基礎的人，或是他已經有 ai, 物聯網 網站等等相關框架運用經驗的人，但還想再嘗試結合即時影音互動的開發者。

這個演講的設計，特別適合那些沒有即時音互動技術概念的人聽，只要他們有 python 開發基礎了，他們真的想用python 去實現各種影音互動的應用，那麼這樣子的就能開始了。這類應用也許是遠端動物照護、遠端教學、遠端陪讀、遠端語音聊天、遠端即時ai運算、遠端醫療等等。只要是對於 python 實現應用有熱情的人，此演講要讓大家滿載而歸，並且回去之後還能實際拿來開發。

如果你是擁有極大的熱情，想要用 python 作一些有助於社會之應用的朋友們，又有開發python程式經驗的人， 你將會是最適合這個演講的聽眾。
