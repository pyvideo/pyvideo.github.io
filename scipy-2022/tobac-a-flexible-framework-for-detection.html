Tobac: A Flexible Framework for Detection

Mon 11 July 2022
By Robert Jackson Bhupendra Raul
  • YouTube

Description

tobac is a Python package to identify, track and analyze objects in different types of gridded datasets, such as model simulations, radar volume scans, and satellite images. tobac is a modular package that includes different algorithms for feature identification, tracking, and analyses. Analysis and visualization methods provide a convenient way to use and display the tracking results.

A critical mass of users, contributors, and maintainers is critical for a successful community code. The project is currently extended (Version 1.5 and 2.0 ) by several contributors to include additional workflows and algorithms using the same structure, syntax, and data formats. Version 2.0 will merge the TINT into tobac, and add more features with Xarray capabilities. tobac will serve as a modular platform for performing Lagrangian analyses and evolve into a sustainable choice for tracking and analyzing atmospheric features.
