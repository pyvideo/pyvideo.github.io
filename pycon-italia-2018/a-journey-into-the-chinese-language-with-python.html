A journey into the Chinese language with Python

Sat 21 April 2018
By Ezio Melotti
  • YouTube

Description

The Chinese language includes over 200 radicals used to form over 40 thousands characters that can be combined to create almost 100 thousands words. Then there are tones, classifiers, homophones, multiple pronunciations for each character, traditional and simplified variants, different kind of romanization… it’s no wonder that Chinese is considered one of the most difficult languages to learn.

However Chinese is actually pretty easy and during this talk you will learn more about this beautiful language and how I’ve been learning it with the help of Python.
