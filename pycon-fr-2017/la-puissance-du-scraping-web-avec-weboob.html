La puissance du scraping web avec Weboob

Sun 24 September 2017
By Laurent Bachelier
  • YouTube

Description

Le "scraping" consiste à récupérer de façon massive des données enfermées sur un site web.

Si Weboob (Web Outside Of Browsers) est surtout connu pour ses applications finales et les modules qu'il fournit, c'est aussi un puissant framework qu'il est possible d'utiliser dans des cas de scraping massif.

Vous verrez comment instrumentaliser Weboob à ces fins.

Une comparaison avec les solutions existantes comme Scrapy sera faite, et les fonctionnalités qui rendent Weboob unique seront abordées.
